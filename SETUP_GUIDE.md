# Setup Guide

## Requirements
- Node.js 18+
- npm

## Installation
```bash
npm install
npm run dev
```

## Deployment
Use Vercel, Netlify, or any static hosting.
```bash
npm run build
```
Upload the `dist` folder.

## Assets
Product images are stored in `public/assets/products/` and the brochure is stored at `public/assets/sun-engineering-brochure.pdf`.

## Customization
Update company information, reviews and product details inside `src/data/products.js`.
