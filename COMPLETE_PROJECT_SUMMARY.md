# Complete Project Summary

## Stack
React, Vite, React Router, Framer Motion, Lucide React, CSS variables.

## File groups
- Configuration: package.json, vite.config.js, tailwind.config.js, postcss.config.js
- Core: main.jsx, App.jsx, index.css
- Context: ThemeContext.jsx
- Components: Navbar, Footer, ProductCard, ReviewCard, FAQAccordion, Chatbot, WhatsAppButton, OrderFlow
- Pages: Home, About, Products, ProductDetail, Services, Gallery, Connect, Login, RequestQuote, FAQs, Order
- Data & utils: products.js, whatsappHelper.js

## Business coverage
Sun Engineering content includes rebar couplers, rebar threading machines, motorized rod cutting machines, steel threaded rods, foundation bolts, thread protection caps and on-site threading services.
